This is a super duper fun quiz with AI, to test your skills :D

## Getting Started

First, run the development server:

```bash
npm run dev
```

Also add .env.local file with This
```
AI_GATEWAY_API_KEY=your_vercel_ai_api_key_here
```

Open [http://localhost:3000](http://localhost:3000) with your browser to test the fun quiz.
